About 3D Document Mind Mapper

This interactive tool transforms text documents into immersive 3D knowledge graphs using natural language processing and force-directed layouts.

Key Features:

  • NLP-powered keyword extraction using compromise.js
  • Interactive 3D force-directed graph visualization
  • VR-style navigation using Device Orientation API
  • Full 3D environment with dynamic camera controls
  • Real-time graph manipulation and customization
  • Local file processing - your data never leaves your device

The application uses Three.js for 3D rendering and WebGL for hardware acceleration, providing a smooth experience even with complex documents.
